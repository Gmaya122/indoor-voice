# Indoor Voice

A Python program that converts SHOUTED text into a calm, lowercase response.

## What it does

Takes any text as input and prints it back entirely in lowercase — because there's no need to shout indoors.

## How to run

```bash
python indoor.py
```

**Example:**
```
HELLO HOW ARE YOU
hello how are you
```

## Concepts used

- User input with `input()`
- String method `.lower()`
